# 에러 코드 명세서

## 코드 체계

```
{카테고리}_{순번}_{플랫폼}
```

| 구성 | 설명 | 예시 |
|------|------|------|
| 카테고리 | 오류 영역 (ui, dat, net, sys) | `ui` |
| 순번 | 4자리 숫자 | `0001` |
| 플랫폼 | `and` (Android) / `ios` (iOS) | `and` |

## 카테고리

| 카테고리 | 코드 접두사 | 설명 |
|----------|-----------|------|
| UI | `ui_` | 화면 렌더링, 위젯 관련 |
| Data | `dat_` | 데이터 로드, 파싱, 저장 |
| Network | `net_` | API 통신, 네트워크 연결 |
| System | `sys_` | 시스템 권한, 디바이스 기능 |

## 에러 코드 목록

### UI 렌더링 (ui)

| 코드 | 설명 | 발생 조건 | 대응 |
|------|------|----------|------|
| `ui_0001_and` | 탭 전환 실패 | IndexedStack 인덱스 범위 초과 | tabIndexProvider 값 검증 |
| `ui_0002_and` | 모달 렌더링 실패 | showModalBottomSheet context 오류 | BuildContext 유효성 확인 |

### 데이터 (dat)

| 코드 | 설명 | 발생 조건 | 대응 |
|------|------|----------|------|
| `dat_0001_and` | Mock 데이터 로드 실패 | MockData 클래스 초기화 오류 | 앱 재시작 |

### 네트워크 (net)

V1은 네트워크 통신을 사용하지 않습니다. API 연동 시 확장 예정.

### 시스템 (sys)

V1에서는 시스템 에러 코드가 정의되지 않았습니다.

## V2 확장 예정

- `net_0001_and`: API 타임아웃
- `net_0002_and`: 인증 토큰 만료
- `sys_0001_and`: 생체인증 실패
