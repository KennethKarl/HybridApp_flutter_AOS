## 주요 기능

### P0 — 필수 (MVP)

- [o] **회원가입 / 로그인**
  - Google 소셜 로그인

- [o] **Tabbar 구조**
  - 5개 탭
  - 탭 간 이동 시 애니메이션
  - Tab1: AI 카메라 분석 화면 (아래 별도 기능 명세 참고)
  - Tab2: AI와 채팅 프로그램
  - Tab3~5: 각각의 WebView 로드

- [o] **Tab1 — AI 카메라 음식 분석** ★★☆
  - 카메라로 음식·명함·상품 촬영 
  - 갤러리에서 사진 선택도 지원 
  - 촬영 이미지를 AI 라이브러리에 전달
  - AI 분석 결과 (이름·설명·특징 등) 화면에 텍스트 출력
  - 분석 중 로딩 인디케이터 표시
  - 오류 시 재시도 버튼 노출
  - 발표 데모: 명함·상품을 카메라로 찍어 즉각적인 AI 반응 시연

## 화면 목록

| 화면 | 설명 |
|------|------|
| 스플래시 | 앱 로딩 |
| 온보딩 | 최초 실행 가이드 (3장) |
| 로그인 | 이메일 / 소셜 로그인 |
| Tab1 — AI 카메라 | 카메라 프리뷰 + 촬영/갤러리 선택 + AI 분석 결과 |
| Tab2~5 — WebView | 각각의 외부 URL WebView 로드 |

---

## 사용자 플로우 (Tab1)

```
앱 실행
  └─ Tab1 진입
       ├─ [카메라 권한 미허용] → 권한 요청 다이얼로그
       │       ├─ 허용 → 카메라 프리뷰 시작
       │       └─ 거부 → 갤러리 선택만 가능
       └─ 카메라 프리뷰 화면
               ├─ [촬영 버튼] → 이미지 캡처
               └─ [갤러리 버튼] → 갤러리 선택 화면 진입
                       └─ Vision API 전송 (로딩 인디케이터)
                               ├─ 성공 → 분석 결과 텍스트 표시
                               └─ 실패 → 오류 메시지 + 재시도 버튼
```

## 수용 기준 (Tab1 AI 카메라)

| ID | 기능 | 수용 기준 |
|----|------|-----------|
| AC-C01 | 카메라 권한 | 최초 진입 시 카메라 권한 요청; 거부 시 갤러리 선택 모드로 fallback |
| AC-C02 | 촬영 | 셔터 버튼 탭 → 1초 이내 이미지 캡처 완료 |
| AC-C03 | 갤러리 선택 | 갤러리 버튼 탭 → 갤러리 선택 화면 진입; 선택 후 분석 자동 시작 |
| AC-C04 | AI 분석 요청 | 이미지 선택/촬영 후 자동으로 Vision API 호출; 로딩 인디케이터 표시 |
| AC-C05 | 분석 결과 표시 | API 응답 수신 후 100ms 이내 텍스트 렌더링; 이름·설명·특징 포함 |
| AC-C06 | 오류 처리 | 네트워크 오류 또는 API 실패 시 오류 메시지 + 재시도 버튼 노출 |
| AC-C07 | 이미지 전처리 | 전송 전 1024×1024 이하로 리사이즈, JPEG 85% 압축 |

## 비기능 요구사항

| 항목 | 기준 |
|------|------|
| 콜드 스타트 시간 | ≤ 2초 |
| API 응답 후 렌더링 | ≤ 100ms |
| AI Vision API 응답 | ≤ 5초 (타임아웃 10초) |
| 최소 지원 OS | Android 8.0 (API 26) / iOS 15.0 |
| 메모리 사용량 | ≤ 200MB (카메라 사용 시) |
| 크래시율 | ≤ 0.1% |
| 오프라인 지원 | 기록 입력은 오프라인 가능, 동기화는 온라인 시 자동 |
| 이미지 전송 크기 | ≤ 1MB (1024×1024 JPEG 85% 압축 후) |

---

## Vision API 연동 명세

| 항목 | 내용 |
|------|------|
| 인증 | API Key (앱 설정 파일 또는 서버 프록시) |
| 이미지 포맷 | Base64 인코딩 JPEG |
| 최대 이미지 크기 | 1024×1024, 85% 압축 |
| 요청 형식 | Multimodal message (image + prompt) |
| 프롬프트 | `"이 사진에 있는 것을 분석해 이름, 설명, 특징을 한국어로 알려줘."` |
| 응답 형식 | 텍스트 (이름 / 설명 / 특징 항목 포함) |
| 타임아웃 | 10초 |
| 재시도 | 최대 1회 자동 재시도 후 오류 UI 노출 |
