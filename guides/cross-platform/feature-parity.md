# 크로스 플랫폼 — Feature Parity (기능 동등성) 추적

> **목적**: Android와 iOS 간 기능 동등성을 추적하여, 한 플랫폼에만 구현된 기능이 누락되지 않도록 관리한다.
> **관리**: 양 플랫폼 개발자 또는 PM이 공동 관리.

---

## 기능 동등성 현황

| 기능 | Android | iOS | 비고 |
|------|---------|-----|------|
| 로그인 | ✅ 완료 | ✅ 완료 | |
| 회원가입 | ✅ 완료 | 🔨 진행 중 | iOS는 TASK-005 |
| 푸시 알림 | ✅ 완료 | 📥 예정 | |
| 생체인증 | 📥 예정 | 📥 예정 | |
| ... | ... | ... | |

### 상태 범례

| 아이콘 | 의미 |
|--------|------|
| ✅ | 구현 완료 |
| 🔨 | 개발 진행 중 |
| 📥 | 예정 (미착수) |
| ❌ | 미지원 (의도적 제외) |

---

## 동등성 갭 이력

| 날짜 | 기능 | 갭 내용 | 해소 계획 |
|------|------|---------|----------|
| YYYY-MM-DD | 회원가입 | iOS 미구현 | iOS TASK-005로 해소 예정 |
